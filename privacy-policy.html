1. Erhebung und Verarbeitung personenbezogener Daten

Diese App erhebt keine personenbezogenen Daten und speichert keine Nutzerdaten auf externen Servern. Alle Einstellungen (z. B. Bundesland, Urlaubstage) werden ausschließlich lokal auf dem Gerät gespeichert.

Diese App verwendet Google AdMob zur Schaltung von Werbeanzeigen. AdMob kann zur Personalisierung von Werbung Gerätekennungen verwenden. Weitere Informationen finden Sie in der Datenschutzerklärung von Google.

4. Feiertagsdaten

Die Feiertagsdaten werden über die OpenHolidays API abgerufen. Dabei werden keine personenbezogenen Daten übermittelt.
